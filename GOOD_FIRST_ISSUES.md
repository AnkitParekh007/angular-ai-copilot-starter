# Good First Issues

These are practical starter tasks for contributors who want to improve Angular AI frontend examples.

1. Add or refresh the 20-second README GIF
2. Add Ask / Plan / Execute state screenshots
3. Add dark/light theme polish improvements
4. Add mobile responsive layout refinements
5. Add another mock MCP tool
6. Improve RAG source card accessibility
7. Extend test coverage beyond helper-level tests
8. Add an error-recovery demo state
9. Add keyboard navigation for session and mode controls
10. Add Angular Material variant
11. Add PrimeNG variant
12. Add Storybook component previews
13. Add animated tool timeline
