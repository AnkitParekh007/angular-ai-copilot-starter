# Good First Issues

These are practical starter tasks for contributors who want to improve Angular AI frontend examples.

1. Add screenshot/GIF to README
2. Add dark/light theme toggle improvements
3. Add mobile responsive layout refinements
4. Add another mock MCP tool
5. Improve RAG source card accessibility
6. Add unit tests for streaming service
7. Add error recovery demo state
8. Add keyboard navigation
9. Add Angular Material variant
10. Add PrimeNG variant
11. Add Storybook component previews
12. Add animated tool timeline
