# Recruiter Review Guide

## 30-second review

- scan the README hero screenshot
- read the one-line pitch
- read **What this proves**

## 3-minute technical review

- inspect `src/app/copilot/models/`
- inspect `src/app/copilot/services/`
- inspect `src/app/copilot/components/`
- inspect `src/app/app.component.ts`
- inspect `WHAT_THIS_PROVES.md`

## Best files to inspect

- `src/app/copilot/models/`
- `src/app/copilot/services/`
- `src/app/copilot/components/`
- `src/app/copilot/mocks/`
- `README.md`
- `WHAT_THIS_PROVES.md`

## What a reviewer should notice

This project treats AI frontend work as a product workflow: context, messages, sources, tool state, approvals, execution status, responsive layout, and explicit mock-only boundaries.
